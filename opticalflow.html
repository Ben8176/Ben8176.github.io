Optical Flow for Station-Keeping



Over my 2023 summer internship at Hood Tech Mechanical, I investigated whether an optical flow system, consisting of a micro-camera running an optical flow algorithm and a laser rangefinder, could be used to allow station-keeping in a GPS-denied environment at 80m for thirty to ninety seconds.

Optical flow refers to the pattern of apparent motion of objects, surfaces, and edges in a visual scene caused by the relative motion between an observer and the scene. The idea behind the investigation was that an optical flow system may be able to track the planar position of a hovering UAV in the case where the GPS position estimate becomes unreliable, due to GPS jamming or otherwise.



In total, I identified, programmed, and tested 3 candidate long-range laser rangefinders and 2 optical flow cameras. Between the sensors themselves, system integration with the UAV, and the localization code on the autopilot, this project was constantly mired with challenges. Most were overcome! My usual day to day work on this project involved implementing different hardware and software hypotheses (eg: use of optics, calibration parameters, firmware changes) and then flight testing them in various wind conditions. I would then capture data from the flight logs and analyze it using MATLAB.




I also set up a MATLAB workflow which would automatically grab relevant data from the flight logs, perform some calculations to get absolute position drift from latitude/longitude, then automatically populate an Excel spreadsheet for each test flight. It was important to automate this procedure, as I eventually did around 100 test flights!

Finally, I wrote a 9 page summary report which included key insights, testing methodology, and challenges encountered. I also wrote many smaller supplemental documents and guides as well as a 27 page continuous log of my activities.
